## Запукс проекта

### Шаг 1: Перейти в папку frontend. Запустить frontend

``` bash
npm run dev
```

### Шаг 2: Перейти в папку backend и запустить backend

``` bash
npm run start:dev
```

### Открыть frontend по ссылке: [http://localhost:5173/](http://localhost:5173/)
